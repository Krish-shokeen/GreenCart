# NODE-JS-PROJECT

GreenCart – Local Sustainable Marketplace

Project Overview

GreenCart is a full-stack eco-commerce platform that promotes sustainable living by connecting local sellers offering organic, handmade, or recycled products with buyers in the same city. The platform encourages conscious consumption through transparent listings and community engagement.


Key Features

●	Product Listings: Sellers can add products with photos, price, and sustainability tags.

●	Search & Filter: Buyers can filter by “organic,” “handmade,” “recycled,” etc.

●	Cart & Checkout: Add to cart and simulate checkout (mock payment gateway).

●	Seller Profiles: View seller reputation, ratings, and location.

●	Community Feedback: Comment and rate purchases for credibility.

Tech Stack

●	Frontend: React.js, Tailwind CSS, Redux Toolkit

●	Backend: Node.js, Express.js

●	Database: MongoDB with Mongoose

●	Storage: Cloudinary for image uploads

●	Authentication: JWT with role-based access (Admin, Seller, Buyer)
